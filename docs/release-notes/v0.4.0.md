## 功能

- 新增 SQLite 存储基建，用于下载任务、阅读缓存索引和运行期缓存等本地数据管理 ([ec525ed](https://github.com/ppxb/jm-boom/commit/ec525ed))。
- 新增运行期 API 缓存，缓存 endpoint discovery、图片域名和阅读 manifest，减少重复网络请求 ([a92bbba](https://github.com/ppxb/jm-boom/commit/a92bbba))。
- 新增加密自动登录配置，支持登录页勾选自动登录和设置页账号配置，启动时重新登录刷新用户资料 ([8ae81bd](https://github.com/ppxb/jm-boom/commit/8ae81bd))。
- 新增运行诊断日志和性能调试日志开关，便于定位阅读器、缓存和下载性能问题 ([827e314](https://github.com/ppxb/jm-boom/commit/827e314))。

## 优化

- 下载任务持久化从 JSON 迁移到 SQLite，并对下载进度写入做节流，降低频繁写盘开销 ([ec525ed](https://github.com/ppxb/jm-boom/commit/ec525ed), [e1eb952](https://github.com/ppxb/jm-boom/commit/e1eb952))。
- 阅读缓存改为 SQLite 索引并后台清理，减少翻页关键路径上的目录扫描和清理阻塞 ([bfb5fff](https://github.com/ppxb/jm-boom/commit/bfb5fff), [feafe36](https://github.com/ppxb/jm-boom/commit/feafe36))。
- 下载页单任务内图片下载改为有限并发，提升整章下载速度 ([26ff175](https://github.com/ppxb/jm-boom/commit/26ff175))。
- 前端列表页、排行、每周推荐和搜索页统一 route search 状态和 Query 默认策略，减少筛选状态和返回行为不一致 ([7ceb346](https://github.com/ppxb/jm-boom/commit/7ceb346))。
- 二级页面统一返回按钮和导航状态处理，避免筛选分页污染返回栈 ([0e39140](https://github.com/ppxb/jm-boom/commit/0e39140), [d56f472](https://github.com/ppxb/jm-boom/commit/d56f472))。
- 全局滚动条统一为 6px 宽、4px 圆角并适配主题，去除滚动条箭头 ([b4e82e3](https://github.com/ppxb/jm-boom/commit/b4e82e3))。
- 阅读页 toolbar 改为点击显示/隐藏，避免鼠标移动时频繁弹出 ([90b7eb9](https://github.com/ppxb/jm-boom/commit/90b7eb9))。
